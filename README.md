# BlueMark FPV

BlueMark FPV is a time-crunched defense hackathon demo for friendly identification of cheap FPV drone video signals.

The demo is a non-lethal deconfliction aid. It will simulate an FPV feed, add an authenticated OSD-style marker, degrade the video, run receiver-side detection, and show a human operator whether the signal appears `Friendly Verified`, `Unknown`, `Corrupted`, or `Possible Spoof`.

## Safety Scope

- Identification aid only.
- Human-in-the-loop warning only.
- No autonomous targeting or engagement logic.
- No weaponization, evasion, or operational deployment guidance.
